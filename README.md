# GO-Promo

Веб-приложение для проведения промо-акций и розыгрышей на основе загруженных чеков.

## Возможности

- Регистрация участников акции
- Загрузка фотографий чеков
- Сохранение данных участников
- Административная панель
- Просмотр всех заявок
- Поиск и фильтрация участников
- Ручное удаление недействительных заявок
- Управление победителями
- Защита административного раздела авторизацией

## Используемые технологии

## Как работает система

1. Пользователь открывает сайт.
2. Заполняет регистрационную форму.
3. Загружает фотографию чека.
4. Данные отправляются на сервер.
5. Информация сохраняется в базе данных.
6. Администратор проверяет заявки через панель управления.
7. После проверки формируется список участников розыгрыша.

## Административная панель

Администратор может:

- Просматривать все заявки
- Искать участников
- Проверять корректность чеков
- Удалять подозрительные заявки
- Управлять списком победителей
